# BANCO_DE_DADOS
O que foi/está sendo feito na disciplina de Banco de Dados


Atividade proposta pelo professor:
Banco de Dados para um Jogo Online
Descreva quais seriam os requisitos necessários para modelar um banco de dados para um jogo online
competitivo, como Valorant, que possibilite o cadastro de jogadores, a participação em partidas, o registro de
estatísticas, a formação de equipes e a consulta de histórico de jogos.
Lembre-se de que os jogadores e os personagens (agentes) devem estar previamente cadastrados no sistema.
É também importante cadastrar as partidas, os mapas, os modos de jogo e os desenvolvedores/publicadoras
responsáveis pelo jogo.
Para descrever esses requisitos, baseie-se na sua experiência jogando ou acompanhando jogos online
competitivos.
Em seguida, modele um Diagrama Entidade-Relacionamento (DER) com base nos requisitos descritos.
Mostre também:
- as cardinalidades mínima e máxima de cada relação
- os tipos de entidades (forte ou fraca)
- os graus de relacionamento
- os tipos de atributos
